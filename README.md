

### Toggletheme: Responsive Theme Switcher
A lightweight, high-performance React component that demonstrates core frontend skills including state management, responsive design, and CSS-in-JS logic.[1]

### Key Features
- **Dynamic Theming:** Seamlessly switch between Light and Dark modes using React Hooks.[2]
- **Fully Responsive:** Optimized for all devices using modern CSS (Flexbox, Viewport units, and Media Queries).[3]
- **Clean Architecture:** Built with functional components and a mobile-first design approach.[4]

### Tech Stack
- **Frontend:** React.js, JavaScript (ES6+)
- **Styling:** CSS3 (Custom Properties, Transitions)
- **Tools:** Vite, Lucide-react (optional for icons)

### Getting Started
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

### Project Impact
This project serves as a "Perfect React Demo" for interviews, highlighting ability to handle UI state, DOM event listeners, and cross-device compatibility.[1]
